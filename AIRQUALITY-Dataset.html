Project Overview

This case study analyzes the airquality dataset, which contains daily air quality measurements in New York from May to September 1973. The goal is to explore patterns in ozone levels, identify high-risk pollution days, and examine relationships with weather variables like temperature and wind.

# Impute missing values with column mean
cleaned <- airquality %>%
  mutate(
    Ozone = ifelse(is.na(Ozone), mean(Ozone, na.rm = TRUE), Ozone),
    Solar.R = ifelse(is.na(Solar.R), mean(Solar.R, na.rm = TRUE), Solar.R)
  )

Monthly Trends in Ozone

avg_ozone_pm <- cleaned %>%
  group_by(Month) %>%
  summarise(
    avg_ozone = mean(Ozone),
    high_risk_days = sum(Ozone > 100),
    avg_wind = mean(Wind)
  )

ggplot(avg_ozone_pm, aes(x = Month, y = avg_ozone, colour = high_risk_days)) +
  geom_smooth(se = TRUE) +
  labs(title = "Average Ozone by Month", x = "Month", y = "Avg Ozone") +
  theme_light()

Insight: Ozone concentrations peak in July and August, with more days crossing the high-risk threshold (>100).

Visual Exploration

Ozone vs Wind

ggplot(cleaned, aes(x = Wind, y = Ozone)) +

Ozone vs Temperature

ggplot(cleaned, aes(x = Temp, y = Ozone)) +
  geom_point(color = "orange", alpha = 0.6) +

Wind + Temp Effects on Ozone

ggplot(cleaned, aes(x = Temp, y = Ozone, color = Wind)) +
  geom_point(size = 2.5, alpha = 0.7) +
  scale_color_viridis_c() +
  geom_smooth(method = "lm", se = FALSE, color = "black") +
  labs(title = "Ozone vs Temp (Colored by Wind)", x = "Temperature (°F)", y = "Ozone", color = "Wind Speed (mph)") +
  theme_minimal()

Conclusion

  • Ozone levels increase during summer, with July and August showing the most high-risk pollution days.
  • There is a clear relationship: ozone rises with higher temperatures and drops with higher wind speeds.
  • Conditions combining heat and calm winds are the most dangerous — highlighting the importance of weather-aware air quality alerts.
