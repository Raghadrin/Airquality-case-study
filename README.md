# Airquality-case-study
Exploratory data analysis of NYC air quality in 1973 using R

## 🔍 Summary
- Analyzed ozone levels and high-risk pollution days
- Explored relationships between ozone, temperature, and wind
- Used `tidyverse`, `ggplot2`, and `GGally` for data cleaning and visualization

## 📁 Files
- `AIRQUALITY-Dataset.Rmd`: Full R Markdown analysis
- `AIRQUALITY-Dataset.html`: Knitted output for easy viewing

## 📊 Tools Used
- R
- tidyverse
- ggplot2
- GGally
